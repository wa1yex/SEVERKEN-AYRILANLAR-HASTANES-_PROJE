# 🏥 Severken Ayrılanlar Hastanesi - Otomasyon Sistemi

Bu proje, bir hastane yönetim sisteminin temel işlevlerini yerine getirmek amacıyla C# ve SQL Server kullanılarak geliştirilmiş bir Windows Form uygulamasıdır.

## 🚀 Özellikler
- **Hasta Paneli:** Randevu alma, kişisel bilgileri güncelleme ve geçmiş randevuları görüntüleme.
- **Doktor Paneli:** Kendisine atanan randevuları görme, duyuruları takip etme ve profil yönetimi.
- **Sekreter Paneli:** Doktor ve branş yönetimi, randevu oluşturma, duyuru yayınlama ve panel izleme.
- **Veritabanı Entegrasyonu:** Tüm veriler SQL Server üzerinde güvenli bir şekilde saklanır.

## 🛠 Kullanılan Teknolojiler
- **Dil:** C#,SQL
- **Framework:** .NET Framework 4.8 (veya senin versiyonun)
- **Veritabanı:** Microsoft SQL Server
- **IDE:** Visual Studio 2022
